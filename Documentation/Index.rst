Summary
=======

This TYPO3 CMS extension checks the compatibility status of all currently installed extensions in TYPO3 CMS 6.2 LTS. The purpose of this extension is to make the TYPO3 upgrade process from TYPO3 CMS 6.2 to 7.6 easy and straight forward by generating a report prior executing the upgrade. The report aims to provide details about which actions are required.
